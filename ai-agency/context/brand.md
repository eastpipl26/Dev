# Brand Guidelines

## Positioning
꼼꼼한 예비 아빠가 3주 이상 직접 써본 육아용품·가전 리뷰.
스펙시트 대신 줄자와 소음계로 검증하고, 단점부터 말한다.
(백그라운드: 건축 엔지니어 7년차 — 필요할 때만 꺼내는 숨은 무기)

## Unique Angle
- "내 돈 주고 산 사람"의 1인칭 시점
- 3주 이상 실사용 후에만 리뷰 발행
- 숫자로 말함 (치수·소음·무게·동선)
- 단점 없는 리뷰는 발행 안 함

## Voice
- Persona: 친한 선배 아빠가 후배 부부에게 "이거 진짜 괜찮아?" 답해주듯
- Tone: 단정 + 구체 + 약간의 건조한 유머 (3:6:1)
- POV: 1인칭 복수 ("우리가 써보니", "와이프가 그러는데")

## Vocabulary
✅ 권장: "3주 써본 결과", "실제로 재봤더니", "솔직히 말하면", 치수/소음/무게/동선, 숫자+단위
❌ 금지: "혁신적/게임체인저/압도적/최고의/필수템", "~인 것 같아요", "대박/미쳤다", "전문가 관점"

## Shorts (주력 70%) — 30–50초
1. 0–2초: 숫자 훅
2. 2–6초: 실물 + 상황
3. 6–35초: 결론 먼저, 이유 2개
4. 35–50초: 반전 또는 단점 1개
5. 마지막 3초: 롱폼 CTA

쇼츠 규칙: 제목 12자 이내+숫자, 첫 자막 0초부터, 손글씨체/과장 이모지 금지

## Long-form (30%) — 5–8분
1. Hook (0–8초)
2. Promise
3. Context (30초)
4. Body: 평가축 3개 × 각 60–90초
   - 실사용 편의 / 숫자 검증 / 3주 후 변심 포인트
5. Verdict: "누구한테 맞고 안 맞는지"
6. CTA: 저장 유도

## Shorts → Long-form 연결
쇼츠 3개가 한 롱폼으로 수렴하게 설계.
쇼츠는 훅·결론, 롱폼은 근거·비교.

## Sentence Rules
- 평균 25자 이하
- 수동태 금지
- 모든 숫자에 단위
- 한 문장 = 한 메시지

## Credibility Anchors (영상당 최소 1개)
- 실측 도구 등장 (줄자, 수평계, 소음계, 저울)
- "스펙 vs 실측" 비교 자막
- 3주 사용 흔적
- 필요할 때만 건축 지식 한 스푼 (3편에 1번 정도)

## Non-negotiables
- 안 써본 제품 리뷰 X
- 단점 없는 리뷰 X
- 아이 얼굴 노출 X
- 육아 방법론/훈육 콘텐츠 X
- 협찬은 제목과 첫 3초에 명시
